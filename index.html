O que é HTML?

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

O que são tags?

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Estrutura básica

< html >

Define a raiz do documento. Engloba todos os outros elementos.

< head >

Compreende as informações do documento que serão interpretadas pelo navegador, antes de carregar a página.

Por exemplo, título do documento, links para folhas de estilo etc.

< title >

Define o título do documento.

< body >

É onde fica todo o conteúdo de texto, imagem e vídeos, em que o usuário vê e interage, nele os conteúdos são estruturados pelas demais tags do HTML.

Tags

Aqui estão algumas tags básicas utilizadas para definir títulos, parágrafos e marcações no texto ao longo do corpo do documento:

< h1 > até < h6 >

Definem títulos no corpo do texto. Vai de 1 até 6, sendo 1 o título mais importante (maior fonte) e 6 o título menos importante (menor fonte).

< p >

Define um parágrafo

< blockquote >

Define uma seção de citação de alguma outra fonte.

< mark >

Destaca o texto.

< small >

Define um texto pequeno.

< i >, < u > e < strong >

< i > pode representar uma parte do texto com uma diferença semântica, na qual a representação tipográfica padrão é no tipo itálico.

< u > é utilizado para sublinhar um espaço definido de um texto.

< strong > dá ao texto uma forte importância e é tipicamente mostrado em negrito.

< hr >

Exibe uma linha horizontal que separa os conteúdos da página.

< sub > e < sup >

< sub > é utilizado para definir textos subscritos.

< sup > é utilizado para definir textos sobrescritos.

< font >

A tag font era usada no HTML 4 para especificar a fonte, o tamanho da fonte e a cor do texto.

Em versões mais recentes, use as propriedades do CSS para estilizar o texto.

< del >

Demonstra parte excluídas do documento.

< abbr >

Define uma abreviação ou acrônimo.

Atributos

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc…

Há dois tipos de atributos no HTML, os globais que são aceitos por todas as tags, como por exemplo: class, id, lang, style e algumas outras que você pode conferir na documentação também existem os específicos que somente algumas tags possuem, como src, disabled, href, label, etc…

Os principais são:

  • class=”NomeDaClasse”
  • id=”NomeDoId”
  • src=”Link ou diretório da mídia"
  • alt=”Texto alternativo”
  • href=”Url”
  • lang=”Linguagem”
  • target=”blank”

Listas

Para criar listas são utilizadas as tags "< ol >", "< ul >" e "< li >".

A tag < ol > é utilizada para criar listas ordenadas. As listas ordenadas são, por padrão, ordenadas por números.

A tag < ul > é utilizada para criar listas desordenadas. As listas desordenadas são organizadas com "pontinhos".

Em ambos os tipos, os items listados devem ser escritos entre a tag < li > (list item)

Ancorar elementos

Com o HTML é possível criar uma espécie de âncora que linka diferentes elementos, sejam dentro de uma mesma página, como índices e botões de voltar, ou entre páginas diferentes, como um botão que leva a um outro site.

Para isso, é utilizada a tag "< a >" junto com o atributo "href":
< a href="referência"> Texto que se refere ao link < /a >

Referência no href

As referências podem ser endereços de email, endereços URL, arquivos ou o nome de um atributo "id=" criado em alguma parte do texto, por exemplo.

O atributo "id=" serve para identificar de forma única um elemento na página HTML e, assim, indicar destino de âncoras. Na construção de um índice, por exemplo, ao criar os tópicos ao longo da página, deve haver o atributo "id=":

< h2 id="primeiro" > Tópico 1 < /h2 >

E no texto do índice, deve-se utilizar a tag < a href="" > com o id indicado após um "#" na referência:

< ul >
< li >
< a href="#primeiro" > Tópico 1 < /a >
< /li >
< /ul >

Dessa maneira, ao clicar no texto do índice, o usuário será direcionado ao tópico na página.
